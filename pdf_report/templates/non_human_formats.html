Fastq
Files ending in 'fastq.gz' are in gzipped Sanger fastq format, and contain the raw sequence data after demultiplexing. Each sample has two files: the file ending in 'R1.fastq.gz' contains the first reads in the pair, the file ending in R2.fastq.gz contains the second reads in the pair.
